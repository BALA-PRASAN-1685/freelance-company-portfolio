# Born to Publish Nexus

Marketing website built with React, TypeScript, Tailwind CSS, and Vite.

## Stack

- Node.js 20+
- React 19
- Tailwind CSS 3.4
- Vite 7

## Project structure

- `src/sections/` — Page sections (Hero, Services, Portfolio, etc.)
- `src/components/ui/` — shadcn/ui components
- `src/hooks/` — Custom hooks
- `public/assets/` — Static images
- `src/App.tsx` — Root React component
- `src/main.tsx` — App entry point

## Commands

From the project root:

```bash
npm run install:app
npm run dev
```

From the `app/` folder:

```bash
npm install
npm run dev
```

The dev server runs at http://localhost:3000
